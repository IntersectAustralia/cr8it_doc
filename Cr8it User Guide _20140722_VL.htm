1.    About Cr8it

Cr8it is [PB1] a file package and publishing tool that enables you to:

1.     pick files

2.     package the files into a data crate

3.     publish the data crate

 

Features of Cr8it include:[PB2] 

      allows you to add information about your data, i.e. metadata

      data crates includes both the data and the metadata

      developed for on the ownCloud environment (owncloud.org)

      helps you meet the requirements of funding agencies

2.    Using[PB3]  Cr8it on ownCloud[PB4] 

The Cr8it tool is available to users as an application when you are logged into the OwnCloud service.
 
     

3.   

Cr8it
 
Add a New Crate


When you are logged into OwnCloud, the Cr8it icon                  will appear on the left side menu.

      Click on the Cr8it icon and the Cr8ite menu will display along the top of the screen

      Click on the  + icon to Add a New Crate

      Enter a name for your New Crate (the Crate Name is a mandatory field)

      Enter a description for your New Crate

      Click Create

      Your new crate will now display on the screen and will also be available in the list of available Crates from the Cr8it menu

4.    Editing the Name of a Crate

      Using the mouse (or the touchpad) highlight the name of the Crate you would like to edit the title of

      Click on  !Rename Crate from the additional menu options that appear on the right-hand side of the Crate name

      Edit or rename the title of the Crate within the Rename Crate box that is displays on the screen

      Click Rename

      Your Crate will now display the amended title

5.    Add Owncloud files and folders to a Crate



 

[PB5] 

       Click on the Files icon    which appears near the top left-hand side of the screen (when logged into OwnCloud).

      Using the mouse (or the touchpad) highlight the name of the Directory, or the name of the file that you want to add to your current Crate

         Click Add to Create  (which appears to the right hand-side of the Directory or file).  Repeat this step as many times as required.

       

      Click on the Cr8it icon    to view the list of directories and/or files in the Crate[PB6] 

         Please note : using the function Add to Crate does not copy or move the files to a Crate, instead you are building an index (or a manifest) of what will be in the crate when you publish it. Adding a file or folder to a Crate is putting a reference reference to the file or folder into the index, these references are similar to Shortcuts on Windows or links on a Mac.  Some people like to think of a crate like an electronic shopping cart. To avoid ambiguity, we like to refer to the references in the crate as Items. Until you download or publish your Crate, in the interim period, the Cr8it app is building an index of the files in your Crate.  When you publish your Crate the list of indexed files, i.e. the contents of the crate,  will autoamtically be copied to the Crate during the downloading / publishing process.  The Crate will reflect the currency of the files at this point of publishing. [PB7] 

       

 

 

6.    Remove a file or directoryan item  from a Crate

 

      Using the mouse (or the touchpad) highlight the name of the file or directory folder item you would like to remove

      Click on  Remove item, from the additional menu options that appear on the right-hand side of the file name

      A Remove Item pop-up box will appear on the screen with the name of the selected file or directory, Click Remove

10.                  Add Data Creators to a Crate via

 

 

 

 

 


      Navigate to the required Crate

 

Select a Data Creator  (to add via automated lookup)

      Enter firstname, or lastname, or firstname and last name into the search box and click the   search symbol. The search results will appear on screen below search box

      From the search results, Click on the symbol next to the name of the Data Creator to be associated with the Crate.

      Repeat above steps to add additional Data Creators

Or

Add New Data Creators  (to add via manual entry)

      Click on the symbol next to Add New Data Creators and a pop-up box for Add Creator will appear on screen

      Enter the name and email address for the data creator you with to associate with the Crate.

      Click Add

      Repeat above steps to add additional Data Creators

11.                  Remove Data Creator from a Crate

      Click the  minus icon next to the name of the Data Creator to be removed

      The Data Creator is no longer visible in the list

11. Add Grant information

 

 

 




      Navigate to the required Crate

 

Select Grants  (to add via automated lookup)

      Enter grant prefix, grant ID, or grant name or keywords into the search box and click the   search symbol. The search results will appear on screen below search box

      From the search results, Click on the symbol next to the required Grant to associate with the Crate.

      Repeat above steps to select additional Grants

Or

Add New Grants  (to add via manual entry)

      Click on the symbol next to Add New Grant and a pop-up box for Add Grant will appear on screen

      Enter the number, submission year, institution, and title of the grant you wish to associate with the Crate.

      Click the Add button

      Repeat above steps to add additional Data Creators

 

12. Remove Grant inforamation from a Crate

      Click the  minus icon next to the name of the Grant to remove

      The Grant is no longer visible in the list

 

13. Download a Crate

      Navigate to the required Crate

      Click the   download button from the top right-hand menu bar

      Click the  zip button and the zip file will download/save to your computer[PB9] 

14. Delete a Crate

 

       Navigate to the required Crate

       Click the   button and a pop-up box will prompt you if there is content in your crate

       Click Delete button
